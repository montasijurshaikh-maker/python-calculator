## ✨ Features
- ➕ Basic: Add, Subtract, Multiply, Divide
- 📐 Scientific: sin, cos, tan, √, x²
- 💱 Currency Converter (12 currencies)
- 🌙 Dark / Light Theme
- 📜 Calculation History
- ⌨️ Keyboard Support
